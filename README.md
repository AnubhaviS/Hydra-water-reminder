Hydra is a clean, minimal, and aesthetic water reminder web app designed to help users stay hydrated and build a healthy daily habit.

The app focuses on simplicity, usability, and a calm wellness-focused design.


##  Features

- Set a daily water intake goal
- Log water consumption with a single click
- Visual progress indicator
- Shows remaining water to reach the goal
- Motivational messages to encourage consistency
- Daily reset option
- Saves progress using localStorage
- Fully responsive and mobile-friendly design


## Design Philosophy

- Soft blue and aqua color palette
- Minimal, modern UI
- Smooth animations and rounded components
- Calm, distraction-free user experience
- Optimized for both desktop and mobile


## Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript  
- LocalStorage (for data persistence)
